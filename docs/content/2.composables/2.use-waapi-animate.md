# useWaapiAnimate

The `useWaapiAnimate` composable uses the Web Animations API for high-performance animations.

## Type Definition

```ts
type WaapiTargets = MaybeRef<VueInstance | HTMLElement | HTMLElement[] | SVGElement | SVGElement[] | NodeList | string | null> | MaybeRef<VueInstance | HTMLElement | HTMLElement[] | SVGElement | SVGElement[] | NodeList | string | null>[]

function useWaapiAnimate(
  target: WaapiTargets,
  options: WAAPIAnimationParams
): WAAPIAnimation
```

### Arguments

- **target**: The DOM element(s) to animate.
- **options**: Object containing keyframes and animation options.

::warning
`useWaapiAnimate` **does not support** JS Objects as targets.
::

## Usage

::code-group
```ts [script]
// Basic usage
const animation = useWaapiAnimate('.box', {
  x: [0, 250],
  opacity: [1, 0.5],
  duration: 800,
  ease: 'linear'
})
```

```vue [template]
<template>
  <div class="box" />
</template>
```
::

## Return Value

Returns a `shallowReactive` object wrapping the animation instance.

::warning
**Do not destructure** the return value to maintain reactivity.
::

```ts
const animation = useWaapiAnimate('.target', options)
animation.play()
```

See [AnimeJS WAAPI documentation](https://github.com/juliangarnier/anime/tree/v4?tab=readme-ov-file#web-animations-api) for more details.
